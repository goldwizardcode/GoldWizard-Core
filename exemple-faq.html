Exemples de FAQ pour Breakdance

Ce document présente différentes façons d'implémenter une FAQ avec Breakdance et le script GoldWizard FAQ.

Note importante : Le script GoldWizard FAQ détecte automatiquement la structure de votre FAQ, vous pouvez donc utiliser n'importe laquelle des structures ci-dessous.

1. Structure recommandée (avec les classes spécifiques)

Cette structure utilise les classes spécifiques du script et est la plus fiable :

<div class="goldwizard-faq-container"> <div class="goldwizard-faq-item"> <div class="goldwizard-faq-question"> Question 1 : Comment fonctionne la FAQ ? </div> <div class="goldwizard-faq-answer"> <div class="goldwizard-faq-answer-inner"> Voici la réponse à la première question. Le script détecte automatiquement cette structure et applique l'animation d'accordéon. </div> </div> </div> <div class="goldwizard-faq-item"> <div class="goldwizard-faq-question"> Question 2 : Puis-je personnaliser les styles ? </div> <div class="goldwizard-faq-answer"> <div class="goldwizard-faq-answer-inner"> Oui, vous pouvez personnaliser tous les styles via Breakdance ou en modifiant le fichier CSS. Les couleurs, espacements et animations sont facilement modifiables. </div> </div> </div> </div>

2. Structure simplifiée avec Breakdance Rich Text

Cette structure est plus simple à créer avec l'éditeur de texte riche de Breakdance :

<div class="goldwizard-faq-container"> <strong>Question 1 : Comment fonctionne la FAQ ?</strong> <p>Voici la réponse à la première question. Le script détecte automatiquement cette structure et applique l'animation d'accordéon.</p> <strong>Question 2 : Puis-je personnaliser les styles ?</strong> <p>Oui, vous pouvez personnaliser tous les styles via Breakdance ou en modifiant le fichier CSS. Les couleurs, espacements et animations sont facilement modifiables.</p> </div>

Dans cette structure, le script détectera automatiquement :

  • Les éléments <strong> comme questions
  • Les paragraphes <p> qui suivent comme réponses

3. Structure avec titres et paragraphes

Vous pouvez également utiliser des titres pour les questions :

<div class="goldwizard-faq-container"> <h3>Question 1 : Comment fonctionne la FAQ ?</h3> <div> Voici la réponse à la première question. Le script détecte automatiquement cette structure et applique l'animation d'accordéon. </div> <h3>Question 2 : Puis-je personnaliser les styles ?</h3> <div> Oui, vous pouvez personnaliser tous les styles via Breakdance ou en modifiant le fichier CSS. Les couleurs, espacements et animations sont facilement modifiables. </div> </div>

Le script détectera :

  • Les titres <h3> comme questions
  • Les <div> qui suivent comme réponses

Instructions d'utilisation dans Breakdance

  1. Créez un élément "Div" ou "Section" dans Breakdance
  2. Ajoutez la classe goldwizard-faq-container à cet élément
  3. À l'intérieur, ajoutez du texte riche ou des éléments HTML en suivant l'une des structures ci-dessus
  4. Le script détectera automatiquement la structure et appliquera les animations
Conseil : Pour vérifier que votre FAQ fonctionne correctement, ouvrez la console du navigateur (F12) et recherchez les messages commençant par "GoldWizard FAQ:". Vous devriez voir des messages indiquant que des questions ont été trouvées.
